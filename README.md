## 项目简介

本项目是一个临床决策支持系统，使用8项常规临床检测指标预测PCI术后患者发生MACE的风险：
- **Age** - 年龄
- **Sex** - 性别
- **BMI** - 体重指数
- **eGFR** - 估算肾小球滤过率 (根据肌酐计算)
- **Glucose** - 血糖
- **Hb** - 血红蛋白
- **RDW** - 红细胞分布宽度
- **WBC** - 白细胞计数

## 功能特性

### 1. 风险预测
- 输入患者基本信息和实验室检查数据
- 实时计算MACE发生风险概率
- 提供风险等级分类（低/中/高）
- 显示各特征的贡献度分析

### 2. 模型对比
- 15种机器学习算法的性能对比
- AUROC对比图表
- ROC曲线可视化
- 校准曲线分析

### 3. 阈值分析
- 多种阈值策略对比（Youden、敏感度优先、NPV优先等）
- 阈值-性能曲线
- 决策曲线分析 (DCA)
- 阈值热力图

### 4. 阈值策略选项
- **Youden指数**: 平衡敏感度和特异度
- **敏感度≥85%**: 适合筛查场景
- **NPV≥95%**: 适合排除诊断
- **成本敏感(10:1)**: 考虑假阴性成本较高的情况

## 技术栈

- **前端**: HTML5, CSS3, JavaScript, Chart.js
- **后端**: Python, Flask
- **模型**: Naive Bayes (NB) - 最终选定的最优单一模型

## 模型选择说明

经过15种机器学习算法的系统对比（包括LightGBM、XGBoost、Random Forest、SVM、Neural Network等），**Naive Bayes (NB)** 被选为最优单一模型。

### 选择理由
- 在测试集和外部验证集上表现稳定
- 模型简单、可解释性强
- 适合临床部署和应用

## 使用指南

### 风险预测
1. 在左侧表单中输入患者的临床数据
2. 选择适合的阈值策略
3. 点击"预测MACE风险"按钮
4. 查看预测结果和临床建议

### 阈值选择建议
- **筛查场景**: 选择"高敏感度"策略，降低漏诊率
- **确诊场景**: 选择"高特异度"策略，降低误诊率
- **平衡场景**: 选择"Youden指数"策略
- **资源有限**: 选择"成本敏感"策略

## 模型性能 (NB最优模型 - 真实实验数据)

### 测试集性能

|------|------|-------|
| AUROC | 0.841 | 0.778-0.899 |
| 敏感度 | 0.865 | - |
| 特异度 | 0.624 | - |
| PPV | 0.242 | - |

### 外部验证集性能
| 指标 | 数值 | 95%CI |
|------|------|-------|
| AUROC | 0.755 | 0.670-0.833 |
| 敏感度 | 0.703 | - |
| 特异度 | 0.721 | - |
| PPV | 0.236 | - |
| NPV | 0.952 | - |
| F1 | 0.329 | - |
| Brier | 0.110 | - |

### 选择标准
NB被选为最优模型的标准：**外部验证集AUROC最高 + 泛化差距最小**
- Test-External Gap: +0.086 (所有模型中最小)
- 优于LightGBM (Gap=+0.229) 等复杂模型

### 推荐阈值
**阈值 = 0.075**
- 测试集: Sens=0.865, Spec=0.673, NPV=0.976
- 外部集: Sens=0.703, Spec=0.721, NPV=0.952

### 特征重要性 (Permutation + SHAP)
1. Glucose (血糖) - 最重要
2. eGFR (肾小球滤过率)
3. WBC (白细胞)
4. BMI (体重指数)
5. Hb (血红蛋白)
6. Age (年龄)
7. RDW (红细胞分布宽度)
8. Sex (性别) - 重要性最低
